## Lecture - 10 Summary 
### Topic : While & do-while loops in C++

There are 3 types of loops in C++:
1. for loop
1. while loop
1. do-while loop

**for loop syntax :**
```c++
    for(intilization; condition ; updation){
    
        loop body(C++ code);
    
    }
```    
**while loop syntax :**
```c++
    initilization;
    while(condition){
    
        loop body(C++ code);

        updation;
        
    }
```
**do-while loop syntax :**
```c++
    initilization;
    do {
    
        loop body(C++ code);

        updation;
        
    } while(condition);
```
